# Accident Detection System

A computer vision project that automatically detects road accidents using deep learning.

## Features
- Real-time accident detection from video footage
- Uses YOLO object detection model
- Trained on custom accident dataset

## Technologies Used
- Python
- OpenCV
- YOLO
- Deep Learning

## Dataset
- Custom accident dataset with labeled images

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the detection script# my_project
